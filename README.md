# Customer Support Ticket Triage System - successfully accomplished the week 1 task

## Project Description
This project is developed as part of Week-1 Internship at Hinelix Technologies.
The objective is to build a rule-based customer support ticket triage system
that classifies issues, assigns priority, calculates SLA, and generates reports
using Python.

## Tools Used
- Python
- Google Colab
- GitHub

## Workflow
- Loaded and explored customer support ticket dataset
- Cleaned ticket messages for processing
- Classified issues into predefined categories
- Assigned priority levels based on urgency
- Calculated SLA hours and due time for each ticket
- Generated management-level reports and exported final dataset

## Output Files
- Week1_Customer_Support_Triage.ipynb
- final_ticket_report.csv
